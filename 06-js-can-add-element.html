Exercise 6

Javascript สามารถเพิ่ม tag html ได้

ลองทำ

  • เปิด Console
  • พิมพ์ let list = document.querySelector('#list'); ลงใน Console
  • พิมพ์ let new_item = document.createElement('li'); ลงใน Console
  • พิมพ์ list.append(new_item); ลงใน Console
  • พิมพ์ new_item.textContent = 'ถ้างงๆ ให้ยกมือ'; ลงใน Console
  • พิมพ์ new_item.style.fontSize = '40px'; ลงใน Console

แบบฝึกหัด

  1. เปิดไฟล์ js-can-add-element.html
  2. แก้ไขปุ่ม โดยเมื่อกดปุ่มแล้วจะเพิ่ม <li> ใหม่เข้ามาใน list นี้

ลับสมอง

  1. แก้ไขปุ่ม โดยเมื่อกดปุ่มแล้วจะเพิ่ม <li> ใหม่เข้ามาใน list นี้ 5 อัน (ลองใช้ for-loop)
