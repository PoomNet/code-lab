Exercise 1

Javascript สามารถ handle event ได้

ตัวอย่าง (เฉพาะที่ใช้บ่อยๆ) ดูเพิ่มเติม

<button onclick="alert('hello world')">alert</button>
<button onclick="console.log('hello-world')">console.log</button>
<input type="text" name="keypress" onkeypress="onKeyPress(event)"> <script> function onKeyPress(event) { console.log(event) console.log('you press key:', event.key) } </script>
<select id="select" name="selct" onchange="onChange(event)"> <option value="a">a</option> <option value="b">b</option> <option value="c">c</option> </select> <script> function onChange(event) { console.log(event) console.log('you select:', event.target.value) } </script>
