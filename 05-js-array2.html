Exercise 5

การจัดการ Array ใน Javascript

Map method - syntax

var new_array = arr.map(function callback(currentValue[, index[, array]]) { // Return element for new_array }[, thisArg])

ตัวอย่างการใช้

ตัวอย่างนี้ใช้งาน map() โดยการคูณ 2 ให้กับค่าทุกค่าใน array

var array1 = [1, 4, 9, 16]; // pass a function to map const map1 = array1.map(x => x * 2); // expected output: Array [2, 8, 18, 32]

แบบฝึกหัด

จงเปิด console ของ Google Chrome ขึ้นมาแล้วทำตามขั้นตอนดังนี้

  1. Copy array นี้ไปไว้ใน console
    var fruits = ['apple', 'banana', 'grapes','8', 'mango', 'orange', '1', '5'];

ลับสมอง

ใช้ map() เพื่อสร้าง array ที่มี element ที่เป็นตัวหนังสือเป็นตัวใหญ่ (capital letters) และ ลบ element ที่เป็นตัวเลขออกจาก array
