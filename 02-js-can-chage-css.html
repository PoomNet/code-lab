Exercise 2

Javascript สามารถเปลี่ยน css ของ DOM ได้

<p> ตัวนี้มี id="hello-world" (ลอง inspect ดูสิ)

ลองทำ

  • เปิด Console
  • พิมพ์ document.querySelector('#hello-world')
  • พิมพ์ document.querySelector('#hello-world').style = 'display: none'; ลงใน Console
  • พิมพ์ document.querySelector('#hello-world').style = 'display: show'; ลงใน Console

แบบฝึกหัด

  1. ลองซ่อนข้อความนี้
  2. แสดงข้อความที่ถูกซ่อน (ในช้อ 1) ให้เป็นสีแดง

ลับสมอง

  1. ลองซ่อน <li> นี้ โดยที่อย่างอื่นต้องเหมือนเดิม
