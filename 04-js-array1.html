Exercise 4

การจัดการ Array ใน Javascript

Filter method - syntax

var newArray = arr.filter(callback(element[, index[, array]])[,thisArg])

ตัวอย่างการใช้

ตัวอย่างนี้ใช้งาน filter() ในการเลือกเฉพาะค่าใน Array ที่มีค่ามากกว่าหรือเท่ากับ 10

function isBigEnough(value) { return value >= 10; } var filtered = [12, 5, 8, 130, 44].filter(isBigEnough);

แบบฝึกหัด

จงเปิด console ของ Google Chrome ขึ้นมาแล้วทำตามขั้นตอนดังนี้

  1. Copy array นี้ไปไว้ใน console
    var fruits = ['apple', 'banana', 'grapes', 'mango', 'orange'];
  2. ใช้ filter() เพื่อหาค้นหาใน array ที่ตรงกับค่า input ที่ใส่เข้ามา
  3. Code เริ่มต้น function filterItems(query) { return fruits.filter( function (item) { /* put your code here */ }) }
  4. เรียก function filterItems() ดังนี้
    filterItems('apple');

ลับสมอง 1

จงทำให้ function filterItems() สามารถค้าหาแบบ partial search (คำใบ้: ใช้ IndexOf())
