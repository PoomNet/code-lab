Exercise 3

Javascript สามารถแก้ไขข้อความใน tag html ได้

<p> ตัวนี้มี id="hello-world" (ลอง inspect ดูสิ)

ลองทำ

  • เปิด Console
  • พิมพ์ document.querySelector('#hello-world')
  • พิมพ์ document.querySelector('#hello-world').innerHTML = 'แก้ไข innerHTML'; ลงใน Console
  • พิมพ์ document.querySelector('#hello-world').textContent = 'แก้ไข textContent'; ลงใน Console

แบบฝึกหัด

  1. ลองเปลี่ยนข้อความนี้เป็น link ไปที่ https://google.com
  2. textContent กับ innerHTML ต่างกันยังไง?

ลับสมอง

  1. ลองเปลี่ยนข้อความนี้เป็นปุ่ม ถ้ากดปุ่มแล้วจะเปลี่ยนสี background ของหน้านี้เป็นสีดำ
